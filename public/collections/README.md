# コレクションカードの背景画像

`/collections` ページの各カードの背景に使う画像です。
以下の4つのファイル名で、このフォルダ（`public/collections/`）に置いてください。

| ファイル名 | コレクション | 今回の写真 |
|---|---|---|
| `snowboard.jpg` | ウィンタースポーツ | スキーリフト／雪山 |
| `summer.jpg` | サマースポーツ | プール／水面 |
| `ball-sports.jpg` | ボールスポーツ | バスケットボール |
| `outdoor.jpg` | アウトドア | 島／山 |

## 推奨仕様
- カードは縦長（3:4）。縦・横どちらの写真でも `object-cover` で中央基準に切り抜かれます。
- 目安サイズ: 横800〜1200px程度（カードが小さいので大きすぎなくてOK）

※ Heroバナーの画像（`public/banners/`）とは別物です。こちらはカード用。

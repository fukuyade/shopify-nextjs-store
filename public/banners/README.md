# Heroバナー画像の置き場所

トップページのHeroカルーセルは、このフォルダの画像を表示します。
以下の **4つのファイル名そのまま** で画像を置いてください（コレクションのhandleに対応）。

| ファイル名 | コレクション | 内容の目安 |
|---|---|---|
| `snowboard.jpg` | ウィンタースポーツ | スノーボードなど冬の代表商品 |
| `summer.jpg` | サマースポーツ | 夏の代表商品 |
| `water.jpg` | ウォータースポーツ | サーフ・SUPなど水上の代表商品 |
| `outdoor.jpg` | アウトドア | 登山・キャンプの代表商品 |

## 推奨仕様
- 形式: JPG（`.jpg`）。PNG等で用意したい場合は教えてください（コードの拡張子を合わせます）。
- 向き: 横長（ランドスケープ）
- サイズ: 横 1600〜1920px / 比率 16:9 程度（例 1600×900, 1920×1080）
- 文字は画像左下に小さく重なるので、左下に重要な被写体を置きすぎないと見やすいです。

画像を4つ置いて `git add . && git commit && git push` すると、Vercelに反映されます。
